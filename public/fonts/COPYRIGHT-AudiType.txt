Audi Type Extended font
=======================

The Audi Type Extended font files (auditype_extendednormal.ttf and
auditype_extendedbold.ttf) are included in this repository for deployment.

font through official Audi channels, you may add the files locally for
development and deployment of this Meet&Greet Audi-themed instance.

How to add the font locally
---------------------------
Font files in this directory: auditype_extendednormal.ttf,
auditype_extendedbold.ttf (lowercase filenames required for Vercel/Linux).
